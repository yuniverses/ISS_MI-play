# 迷玩｜同杯遊戲室

一個即時畫圖猜題遊戲平台，玩家可以透過手機進入房間，進行畫圖與猜題的互動遊戲。

## 功能特色

- 🎨 **即時畫圖同步**：畫畫者的筆觸會即時同步給所有猜題者
- 🎯 **猜題競賽**：猜題者輸入答案，答對即可得分
- 📊 **即時排行榜**：顯示當前房間內玩家分數排名
- ⏱️ **30秒倒計時**：每回合30秒，時間到自動切換
- 🎮 **回合制遊戲**：玩家輪流當畫畫者
- 📱 **手機版優化**：針對手機觸控操作優化

## 技術棧

- **前端**：React + Vite + Socket.IO Client
- **後端**：Node.js + Express + Socket.IO
- **即時通訊**：WebSocket (Socket.IO)

## 快速開始

### 安裝依賴

```bash
# 安裝所有依賴（根目錄、伺服器、客戶端）
npm run install-all
```

### 啟動開發伺服器

```bash
# 同時啟動前端和後端
npm run dev
```

或者分別啟動：

```bash
# 終端 1：啟動後端伺服器（端口 3001）
npm run server

# 終端 2：啟動前端開發伺服器（端口 5173）
npm run client
```

### 訪問應用

打開瀏覽器訪問：`http://localhost:5173`

## 使用說明

1. **進入遊戲**
   - 輸入你的暱稱
   - 點擊「進入遊戲」

2. **畫畫者模式**
   - 第一個進入的玩家會自動成為畫畫者
   - 頂部會顯示題目（只有畫畫者可見）
   - 在畫布上畫圖，其他人會看到你的筆觸
   - 使用底部工具列調整顏色、筆刷大小
   - 可以清除畫布重新開始

3. **猜題者模式**
   - 看到畫畫者的即時筆觸
   - 在底部輸入框輸入你的猜測
   - 答對即可得分（越快答對分數越高）
   - 畫畫者被猜到也會得分

4. **回合機制**
   - 每回合30秒
   - 時間到或有人答對後，會輪到下一位玩家當畫畫者
   - 所有玩家輪流畫圖

## 設計特色

- **綠白配色**：使用品牌綠色 (#17B26A) 和白色作為主色調
- **手機優先**：針對手機觸控操作優化，支持手勢繪圖
- **即時同步**：筆觸即時傳輸，低延遲體驗
- **簡潔介面**：排行榜置頂，操作區置底，畫布居中
